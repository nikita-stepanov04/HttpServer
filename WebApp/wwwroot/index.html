Server Information

About the Server

This server is designed to handle various HTTP requests and provides a range of functionalities to support web applications.

Supported Methods

  • GET
  • POST

Status Codes

  • 200 OK
  • 204 NoContent
  • 400 BadRequest
  • 404 NotFound
  • 405 MethodNotAllowed
  • 500 InternalServerError
  • 505 HttpVersionNotSupported

Response types

  • JSON
  • Static files
  • Razor Pages
